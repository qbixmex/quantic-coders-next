# Quantic Coders Blog

## Installation 💻

```bash
# NPM
npm install

# YARN
yarn install

# PNPM
pnpm install

# BUN
bun install
```

## Run Development Mode 🏃‍♂️‍➡️

```bash
# NPM
npm run dev

# YARN
yarn dev

# PNPM
pnpm dev

# BUN
bun dev
```

## Check for errors 🪲

```bash
# NPM
npm run lint

# YARN
yarn lint

# PNPM
pnpm lint

# BUN
bun lint
```

## Build the project 🏗️

```bash
# NPM
npm run build

# YARN
yarn run build

# PNPM
pnpm run build

# BUN
bun run build
```

## Preview pre-build project 👀

**Use this command only if you previously build the project.** 🗒️

```bash
# NPM
npm run preview

# YARN
yarn preview

# PNPM
pnpm preview

# BUN
bun preview
```
